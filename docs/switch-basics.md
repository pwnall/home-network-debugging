# Router Basics: TRENDnet TEG-3102WS

## Model
- **Name:** TRENDnet TEG-3102WS (10-Port Multi-Gig Web Smart Switch)

## Manuals
- **User's Guide:** [Download PDF](https://www.trendnet.com/download_manager/default.asp?iFile=43001)
- **Support Page:** [TRENDnet Support Detail](https://www.trendnet.com/support/support-detail.asp?prod=110_TEG-3102WS)

## High-Level Entry Points
- **Web Interface:** Accessible via HTTP on port `80`. The interface uses a React-based frontend communicating with backend `/api` endpoints.
- **Command Line Interface (CLI):** Accessible over the network via SSH on port `22` (Telnet is disabled). It can also be accessed via a physical console port (RJ-45 to RS-232 serial connection).
